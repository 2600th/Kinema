# Lessons

- Keep `package-lock.json` generated by npm; avoid manually editing platform-specific optional package entries (for example Biome CLI variants), because missing `version` metadata can break CI installs with `Invalid Version`.
- For iPhone/iPad browser issues, prefer `visualViewport`-driven resize handling plus `orientationchange` follow-up resizing, and default Apple mobile browsers to the compatibility WebGL renderer path unless there is a verified reason to use the experimental renderer.
